# ☕ Java Projects by Joe Habre

![Build](https://github.com/Joeehabre/Java-Projects/actions/workflows/build.yml/badge.svg)

Welcome to my collection of Java projects — a showcase of my Computer Science journey at AUB and my passion for building clean, functional, and interactive software.  
These projects explore core programming concepts like **object-oriented design, file handling, GUI development, CLI tools, and problem-solving**.  

Each application is written with **readability, modularity, and future expansion** in mind.

---

## 🧩 Featured Projects

### 🔹 [Study Planner](./study-planner) 📚
A comprehensive command-line planner to help students organize their study tasks.  
Features include task priorities, due dates, tags, filters, sorting, stats, and CSV import/export with autosave.  
Overdue tasks are highlighted in red, and all output is ANSI-colored for readability.  
**Concepts:** OOP, Enums, File I/O (CSV), Data Persistence, CLI UX, Modular Design

---

### 🔹 [Bank Account System](./bank-account) 🏦
A command-line banking simulation supporting multiple accounts with full transaction history.  
Create accounts, deposit, withdraw, transfer between accounts, and review a timestamped log of every transaction. Data persists across sessions.  
**Concepts:** OOP, Encapsulation, File Persistence, Collections, Input Validation

---

### 🔹 [Calculator](./calculator) ➗
A CLI calculator with calculation history, result chaining, and extended operations.  
Type `ans` to reuse the last result, `sqrt` for square roots, `h` to view history, and chain multiple calculations in one session.  
**Concepts:** Control Flow, Exception Handling, Deque, Method Structure

---

### 🔹 [To-Do List](./todo-list) ✅
A text-file-backed task manager with full task lifecycle support.  
Add, edit, remove, and toggle tasks done/undone with `[x]` / `[ ]` display. Clear all completed tasks at once. Progress counter shown at all times.  
**Concepts:** File I/O, ArrayLists, Input Handling, State Persistence

---

### 🔹 [Snake Game](./snake-game) 🐍
The classic Snake game rebuilt with **Java Swing** and polished visuals.  
Features a live high score, gradient snake rendering, a PAUSED overlay, food that never spawns on the snake, WASD + arrow key controls, and a game-over screen showing your score and best.  
**Concepts:** Java GUI (Swing), Timers, Event Handling, Collision Logic, Graphics2D

---

### 🔹 [Expense Tracker](./expense-tracker) 💰
A terminal-based expense tracker for logging and analyzing personal spending.  
Add, delete, and filter expenses by category (partial match) or month/year. View category summaries with a grand total. All data is saved to CSV with proper escaping.  
**Concepts:** File I/O (CSV), OOP, Stream API, Collections, Date Filtering
